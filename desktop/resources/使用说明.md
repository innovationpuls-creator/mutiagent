# OneTree Windows 便携版使用说明

## 运行环境

- Windows 10/11 x64。
- 运行 AI 功能时需要可用的互联网连接。
- 无需安装 Docker、Python、Node.js 或 PostgreSQL。

## 启动

1. 完整解压 `11017810作品.zip`，不要在压缩包预览窗口中直接运行。
2. 双击压缩包根目录中的 `OneTree.exe`。
3. 首次启动会自动初始化本地数据库。桌面窗口出现前不要重复双击。
4. OneTree 窗口出现后即可登录使用。

## 演示账号

- 账号：`demo@mutiagent.local`
- 密码：`demo123456`

## 关闭

直接关闭 OneTree 桌面窗口。程序会自动停止后端、Worker 和本地数据库。

## 数据和日志

- 用户数据：`%APPDATA%\OneTree`
- 运行日志：`%APPDATA%\OneTree\logs`

再次打开 `OneTree.exe` 时会继续使用原有本地数据。

## 启动失败

若程序提示端口占用，请关闭占用 `8000` 或 `55432` 端口的程序，再重新打开 `OneTree.exe`。若仍然失败，请将 `%APPDATA%\OneTree\logs` 中的日志交给开发者检查。
