# 间距系统

> 间距以 4px 为基础单位，但整体使用 Headspace meditation 的慷慨留白。所有间距必须来自 `--space-*` token。

---

## Spacing Scale

| Token | 值 | 用途 |
| --- | --- | --- |
| `--space-4` | `4px` | 边线内缩、进度条内 padding |
| `--space-8` | `8px` | 图标与文字、小元素间距 |
| `--space-12` | `12px` | 紧凑控件内部间距 |
| `--space-16` | `16px` | 按钮垂直 padding、列表间距 |
| `--space-24` | `24px` | 卡片基础 padding、表单组 |
| `--space-32` | `32px` | 常规卡片、模块内大间距 |
| `--space-40` | `40px` | 疗愈卡片、重点控件 |
| `--space-48` | `48px` | 区块内部段落 |
| `--space-64` | `64px` | Section 基础留白 |
| `--space-80` | `80px` | 大屏重点区块 |
| `--space-96` | `96px` | Hero 与大型过渡 |
| `--space-120` | `120px` | 桌面端首屏大留白 |

禁止写没有 token 的 `px` 间距。需要微调也要先定义 token，不能在组件里临时写值。

---

## Section Padding

| Token | Mobile | Tablet | Desktop | 用途 |
| --- | --- | --- | --- | --- |
| `--section-padding-x` | `var(--space-24)` | `var(--space-40)` | `var(--space-64)` | 页面左右安全距离 |
| `--section-padding-y` | `var(--space-48)` | `var(--space-64)` | `var(--space-80)` | 常规 section |
| `--section-padding-y-lg` | `var(--space-64)` | `var(--space-96)` | `var(--space-120)` | Hero、重点插画 |
| `--container-narrow` | `640px` | 长文阅读 |
| `--container-default` | `960px` | 主内容 |
| `--container-wide` | `1120px` | 多列布局 |

```css
:root {
  --section-padding-x: var(--space-24);
  --section-padding-y: var(--space-48);
  --section-padding-y-lg: var(--space-64);
}

@media (min-width: 768px) {
  :root {
    --section-padding-x: var(--space-40);
    --section-padding-y: var(--space-64);
    --section-padding-y-lg: var(--space-96);
  }
}

@media (min-width: 1280px) {
  :root {
    --section-padding-x: var(--space-64);
    --section-padding-y: var(--space-80);
    --section-padding-y-lg: var(--space-120);
  }
}
```

---

## Gap Tokens

| Token | 值 | 用途 |
| --- | --- | --- |
| `--gap-xs` | `var(--space-8)` | 图标、状态点 |
| `--gap-sm` | `var(--space-16)` | 紧凑控件 |
| `--gap-md` | `var(--space-24)` | 按钮组、卡片内分组 |
| `--gap-lg` | `var(--space-40)` | 卡片网格 |
| `--gap-xl` | `var(--space-64)` | 核心板块 |

---

## 应用范例

```css
.btn {
  min-block-size: 44px;
  padding: var(--space-16) var(--space-40);
  gap: var(--gap-xs);
}

.wellness-card {
  padding: var(--space-40);
  display: grid;
  gap: var(--gap-md);
}

.section {
  padding-block: var(--section-padding-y);
  padding-inline: var(--section-padding-x);
}
```

---

## 移动端规则

- 移动端仍然使用 token，不缩成任意小值。
- 主要按钮触控区域不小于 `44px`。
- 卡片内 padding 最低使用 `--space-24`；只有密集列表可使用 `--space-16`。
- section 之间必须保留明显呼吸感，不能硬贴。
