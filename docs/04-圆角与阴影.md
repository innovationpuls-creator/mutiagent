# 圆角 & 阴影

> **核心哲学**：完全拒绝任何锐角和硬边缘。在这个系统中，一切事物都应当是圆润的、像卵石一样的。阴影不再是冰冷的黑灰色，而是散发着温暖环境光的彩色弥散晕轮。

---

## Border Radius Scale (全大圆角系统)

废弃常见的 4px / 6px / 8px 小圆角，直接从 16px 起步。

| Token | 值 | 质感描述 | 适用场景 |
|-------|-----|---------|----------|
| `--radius-sm` | `16px` | **基础微卵石** | 输入框、较小的内层标签。 |
| `--radius-md` | `24px` | **卵石形** | 常规卡片、列表项背景。 |
| `--radius-lg` | `32px` | **大型鹅卵石** | 巨大的内容区块背景、模态对话框。 |
| `--radius-full` | `9999px` | **胶囊/药丸 (Pill)** | **所有按钮**、Badge、头像、圆弧顶部插画。 |

> **绝对规则**：
> 1. 所有的 `<button>` 和主操作入口，必须使用 `--radius-full`，做成标准的 Pill 胶囊形状。
> 2. 不得在界面中出现小于 16px 的圆角，不要试图保留任何锋利的折角。

---

## Shadow Scale (彩色环境光投影)

Headspace 的阴影绝对不是 `#000` 加上一点透明度。在桃色底色 (`#FFE2C5`) 上，阴影必须带有珊瑚色/橘色的温度，使其看起来像是在暖色桌面上投射的光影。

| Token | 值 | 质感描述 | 用途 |
|-------|-----|---------|------|
| `--shadow-sm` | `0 4px 12px oklch(76% 0.12 55 / 0.12)` | **极柔微光** | 静止卡片的基底阴影、输入框。 |
| `--shadow-md` | `0 12px 32px oklch(76% 0.12 55 / 0.18)` | **浮木托起** | **卡片 Hover 时的阴影**，展现强烈的浮力感。 |
| `--shadow-lg` | `0 24px 64px oklch(76% 0.12 55 / 0.25)` | **深远弥散** | 模态框、核心焦点插画的底层光晕。 |
| `--shadow-glow` | `0 0 32px var(--color-primary)` | **品牌光环** | 在极少数需要极致高光的地方使用（如冥想开启按钮），透明度极高。 |

> **解析**：`oklch(76% 0.12 55)` 正是我们的 `--color-primary` (暖珊瑚色)。我们在阴影中直接混合了品牌色，而不是冷灰色。

## 交互态与层级 (Interactions)

### 慵懒悬浮 (Lazy Hover)

如 `07-motion-physics.md` 所述，Hover 时仅仅是轻微放大，但通过 `--shadow-md` 释放巨大的浮力。

```css
.wellness-card {
  background: var(--color-surface);
  border-radius: var(--radius-md); /* 24px */
  box-shadow: var(--shadow-sm);
  /* 从 07-motion-physics 获取的缓动参数 */
  transition: transform var(--duration-lazy-hover) var(--ease-lazy),
              box-shadow var(--duration-lazy-hover) var(--ease-lazy);
}

.wellness-card:hover {
  transform: scale(1.01) translateY(-2px);
  /* 投影扩大，颜色变深但依然是暖调 */
  box-shadow: var(--shadow-md); 
}
```

### 胶囊按钮 (Pill Button)

```css
.btn-primary {
  /* 强制胶囊形状 */
  border-radius: var(--radius-full);
  background: var(--color-primary);
  /* 按钮往往自身就带有柔和的发光阴影 */
  box-shadow: 0 8px 24px oklch(76% 0.12 55 / 0.3);
}
```
