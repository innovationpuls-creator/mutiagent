# 字体系统

> 标题与正文均使用 LXGW WenKai（霞鹜文楷），已安装为系统字体。中文界面专用。

---

## Font Family

```css
/* 标题与正文使用相同字体栈，禁止任何形式的字体降级 */
--font-heading: 'LXGW WenKai';
--font-body:    'LXGW WenKai';
--font-mono:    'LXGW WenKai Mono';
```

| 字体 | 来源 | 格式 | 用途 |
|------|------|------|------|
| LXGW WenKai | 系统字体（brew install） | TTF | 标题、正文、UI 文字 |
| LXGW WenKai Mono | 系统字体（brew install） | TTF | 代码块、终端、技术内容 |

> **强制约束**：系统绝对禁止任何形式的字体降级（如 PingFang SC、Microsoft YaHei 等回退字体）。必须严格保持上述唯一的字体配置。

---

## Font Weight

| Token | 值 | 用途 |
|-------|-----|------|
| `--font-weight-light` | `300` | 极淡提示文字、装饰性辅助文字 |
| `--font-weight-regular` | `400` | 正文、段落、H4-H6 标题、按钮默认态 |
| `--font-weight-medium` | `500` | H1-H3 标题、按钮主要态、强调文字 |

> **强制约束**：Web 的所有内容必须且只能使用上述三种字重（300, 400, 500）。LXGW WenKai 仅提供 Light/Regular/Medium 三个字重，无 Bold/SemiBold。需要加粗强调时使用 Medium 500。绝对禁止使用其他字重或浏览器合成加粗，禁止任何形式的字体降级现象。

---

## Heading Scale（标题字阶）

使用 `clamp()` 实现流式缩放，在移动端和桌面端之间平滑过渡。

| Token | 字号 | 行高 | 字重 | 用途 |
|-------|------|------|------|------|
| `--text-display` | `clamp(3rem, 1rem + 7vw, 6rem)` (48–96px) | `1.1` | `500` | Hero 大标题、落地页主视觉 |
| `--text-h1` | `clamp(2rem, 1.5rem + 2.5vw, 2.5rem)` (32–40px) | `1.2` | `500` | 页面主标题 |
| `--text-h2` | `clamp(1.6rem, 1.2rem + 2vw, 2rem)` (25.6–32px) | `1.3` | `500` | 区块标题、章节标题 |
| `--text-h3` | `clamp(1.25rem, 1.1rem + 0.8vw, 1.5rem)` (20–24px) | `1.35` | `500` | 卡片标题、功能标题 |
| `--text-h4` | `clamp(1.1rem, 1rem + 0.5vw, 1.25rem)` (17.6–20px) | `1.4` | `400` | 子章节标题 |
| `--text-h5` | `clamp(1rem, 0.95rem + 0.25vw, 1.1rem)` (16–17.6px) | `1.4` | `400` | 列表标题、小节标题 |
| `--text-h6` | `clamp(0.9rem, 0.85rem + 0.2vw, 0.95rem)` (14.4–15.2px) | `1.4` | `400` | 最小标题、标签组标题 |

## Body & Caption

| Token | 字号 | 行高 | 字重 | 用途 |
|-------|------|------|------|------|
| `--text-body` | `1rem` (16px) | `2` | `400` | 正文段落、课程描述 |
| `--text-body-relaxed` | `1rem` (16px) | `2.2` | `400` | 长文阅读、文章正文 |
| `--text-body-compact` | `1rem` (16px) | `1.6` | `400` | 表格内文字、密集信息区 |
| `--text-body-sm` | `0.9rem` (14.4px) | `1.8` | `400` | 辅助正文、侧栏说明 |
| `--text-caption` | `0.82rem` (13.1px) | `1.6` | `400` | 时间戳、来源标注、表头 |
| `--text-overline` | `0.72rem` (11.5px) | `1.4` | `500` | 区块标签、分类标签（大写） |
| `--text-button` | `0.95rem` (15.2px) | `1` | `500` | 按钮文字 |

## Line Height 规则

| 场景 | 值 | 理由 |
|------|-----|------|
| 标题（H1-H3） | `1.2 – 1.35` | 标题需要紧凑，突出视觉冲击 |
| 标题（H4-H6） | `1.4` | 小标题略宽松，可读性优先 |
| 正文段落 | `2` | 中文长文需要宽松行距，避免视觉疲劳 |
| 按钮文字 | `1` | 单行文字用 `line-height: 1` + padding 控制高度 |
| 代码块 | `1.7` | 等宽字体行间需要呼吸空间 |

## CSS 使用示例

```css
h1 { font-size: var(--text-h1); font-weight: 500; line-height: 1.2; }
h2 { font-size: var(--text-h2); font-weight: 500; line-height: 1.3; }
h3 { font-size: var(--text-h3); font-weight: 500; line-height: 1.35; }
h4 { font-size: var(--text-h4); font-weight: 400; line-height: 1.4; }
h5 { font-size: var(--text-h5); font-weight: 400; line-height: 1.4; }
h6 { font-size: var(--text-h6); font-weight: 400; line-height: 1.4; }

body {
  font-family: var(--font-body);
  font-size: var(--text-body);
  font-weight: 400;
  line-height: 2;
}

.caption {
  font-size: var(--text-caption);
  color: var(--color-text-secondary);
}

code, pre {
  font-family: var(--font-mono);
  line-height: 1.7;
}
```

## 字体加载策略

### font-display

自托管 WOFF2 字体必须使用 `font-display: swap`，避免 FOIT（不可见文本闪烁）：

```css
@font-face {
  font-family: 'LXGW WenKai';
  src: url('/fonts/lxgw-wenkai-regular.woff2') format('woff2');
  font-weight: 400;
  font-display: swap;
}
```

### 字体预加载

仅预加载首屏必需的字重（Regular 400），其余延迟加载：

```html
<link rel="preload" href="/fonts/lxgw-wenkai-regular.woff2" as="font" type="font/woff2" crossorigin>
```
