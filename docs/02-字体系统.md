# 字体系统

> 项目只使用 LXGW WenKai。文字要温暖、友好、松弛，但不引入额外手写字体或外部字体。

---

## Font Family

```css
:root {
  --font-heading: 'LXGW WenKai';
  --font-body: 'LXGW WenKai';
  --font-mono: 'LXGW WenKai Mono';
}
```

| 字体 | 来源 | 用途 |
| --- | --- | --- |
| LXGW WenKai | `@fontsource/lxgw-wenkai` 或本地自托管 | 标题、正文、UI 文字 |
| LXGW WenKai Mono | 本地自托管 | 代码块、终端、技术内容 |

禁止引入 Google Fonts、Quicksand、Nunito、Caveat、Inter、Roboto、Arial 或系统字体作为显示字体。情绪强调通过颜色、留白和排版节奏实现，不通过额外手写字体实现。

---

## Font Weight

| Token | 值 | 用途 |
| --- | --- | --- |
| `--font-weight-light` | `300` | 极淡提示、装饰性辅助文字 |
| `--font-weight-regular` | `400` | 正文、段落、普通控件 |
| `--font-weight-medium` | `500` | H1-H3、主按钮、强调文字 |

LXGW WenKai 仅使用 Light / Regular / Medium。禁止 Bold、SemiBold 和浏览器合成加粗。

---

## Type Scale

字体大小不使用 viewport width 自动缩放。需要响应式时，在断点里切换 token，而不是写 `vw` 字号。

| Token | Desktop | Mobile | 行高 | 字重 | 用途 |
| --- | --- | --- | --- | --- | --- |
| `--text-display` | `5rem` | `3rem` | `1.1` | `500` | Hero 大标题 |
| `--text-h1` | `2.5rem` | `2rem` | `1.2` | `500` | 页面标题 |
| `--text-h2` | `2rem` | `1.625rem` | `1.3` | `500` | 区块标题 |
| `--text-h3` | `1.5rem` | `1.25rem` | `1.35` | `500` | 卡片标题 |
| `--text-h4` | `1.25rem` | `1.125rem` | `1.4` | `400` | 子章节标题 |
| `--text-h5` | `1.0625rem` | `1rem` | `1.4` | `400` | 列表标题 |
| `--text-h6` | `0.9375rem` | `0.875rem` | `1.4` | `400` | 最小标题 |

```css
:root {
  --text-display: 5rem;
  --text-h1: 2.5rem;
  --text-h2: 2rem;
  --text-h3: 1.5rem;
  --text-h4: 1.25rem;
  --text-h5: 1.0625rem;
  --text-h6: 0.9375rem;
}

@media (max-width: 767px) {
  :root {
    --text-display: 3rem;
    --text-h1: 2rem;
    --text-h2: 1.625rem;
    --text-h3: 1.25rem;
    --text-h4: 1.125rem;
    --text-h5: 1rem;
    --text-h6: 0.875rem;
  }
}
```

## 排版规则

- 中文正文默认 `line-height: 2`，长文可用 `2.15`。
- 标题不使用负字距，`letter-spacing` 保持 `0`。
- 按钮文字使用 `line-height: 1`，高度由 padding 控制。
- 所有长段落使用 `text-wrap: pretty`。

```css
body {
  font-family: var(--font-body);
  font-size: var(--text-body);
  font-weight: var(--font-weight-regular);
  line-height: 2;
  color: var(--color-text-primary);
}

h1, h2, h3 {
  font-family: var(--font-heading);
  font-weight: var(--font-weight-medium);
  letter-spacing: 0;
  text-wrap: pretty;
}
```

## 字体加载

```css
@font-face {
  font-family: 'LXGW WenKai';
  src: url('/fonts/lxgw-wenkai-regular.woff2') format('woff2');
  font-weight: 400;
  font-display: swap;
}
```

只预加载首屏必需字重。其余字重按需加载，避免阻塞首屏。
