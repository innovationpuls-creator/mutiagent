# 暗色模式

> 暗色模式不是纯黑界面。它是温暖、安静、包裹感强的深鸭蓝夜间环境。

---

## 切换方式

- 支持 `prefers-color-scheme: dark`。
- 支持 `<html data-theme="dark">` 手动切换。
- 主题切换不直接动画颜色。需要柔和过渡时，用页面遮罩层做 `opacity` crossfade。

## 暗色阴影

暗色模式中阴影仍然多层，但色相转向深鸭蓝。不要使用白色外发光模拟阴影。

| Token | Dark 值 |
| --- | --- |
| `--shadow-sm` | `0 1px 2px oklch(8% 0.03 235 / 0.42), 0 8px 24px oklch(8% 0.03 235 / 0.28)` |
| `--shadow-md` | `0 2px 6px oklch(8% 0.03 235 / 0.48), 0 18px 48px oklch(8% 0.03 235 / 0.36)` |
| `--shadow-lg` | `0 4px 12px oklch(8% 0.03 235 / 0.54), 0 32px 80px oklch(8% 0.03 235 / 0.44), 0 64px 120px oklch(8% 0.03 235 / 0.32)` |

## CSS 映射

```css
@media (prefers-color-scheme: dark) {
  :root {
    color-scheme: dark;
    --color-background: oklch(16% 0.035 235);
    --color-background-warm: oklch(18% 0.04 235);
    --color-surface: oklch(21% 0.04 235);
    --color-surface-raised: oklch(25% 0.045 235);
    --color-surface-inset: oklch(13% 0.035 235);
    --color-border: oklch(38% 0.04 235);
    --color-text-primary: oklch(92% 0.025 75);
    --color-text-secondary: oklch(80% 0.025 75);
    --color-text-muted: oklch(64% 0.02 235);
  }
}

html[data-theme='dark'] {
  color-scheme: dark;
}
```

---

## Reduced Motion

主题切换、夜间光晕、暗色背景呼吸动画都必须尊重 `prefers-reduced-motion`。降级后直接切换状态，不做位移、缩放或循环动画。
