# 暗色模式 (Night Mode)

> **核心哲学**：Headspace 的暗色模式绝对不是纯黑 `#000` 或冷峻的深灰色。它是一种**深邃且包裹感极强的暖深水鸭蓝 (Warm Dark Teal)**。这种颜色能让你联想到安静的夜晚和深沉的呼吸，而不是冷冰冰的代码编辑器。

---

## Color Scheme 切换

通过 `prefers-color-scheme` 自动检测，或在 `<html>` 上挂载 `data-theme="dark"` 手动切换。

---

## 核心 Token 对照表

### 背景与画布 (The Night Sky)

在暗色模式下，底色变成了深邃的鸭蓝，而不是传统的深灰色。

| Token | 亮色模式 (Day) | 暗色模式 (Night) | 质感描述 |
|-------|----------|----------|---------|
| `--color-background` | `oklch(93% 0.05 70)` 暖桃色 | `oklch(25% 0.04 240)` `#1A2F3A` | **夜空深蓝** |
| `--color-surface` | `oklch(96% 0.03 70)` 浅桃色 | `oklch(30% 0.05 240)` `#1B3A47` | **深水鸭蓝** (稍亮一层) |
| `--color-surface-pure` | `oklch(98% 0.01 70)` | `oklch(35% 0.06 240)` | 用于高亮卡片 |
| `--color-border` | `oklch(85% 0.03 70)` | `oklch(40% 0.06 240)` | 暗色分割线 |

### 文字与墨水 (Starlight Ink)

暗色背景上的文字不能是纯白（太刺眼），应当带有一点点桃色的温度，就像星光一样。

| Token | 亮色模式 | 暗色模式 | 质感描述 |
|-------|----------|----------|---------|
| `--color-text-primary` | 深鸭蓝 | `oklch(96% 0.03 70)` `#FFEDD5` | **星光桃 (Starlight Peach)** |
| `--color-text-secondary` | 次级鸭蓝 | `oklch(80% 0.02 70)` | 稍微暗淡的次要文字 |
| `--color-text-muted` | 浅灰蓝 | `oklch(60% 0.02 240)` | (保持不变) |
| `--color-text-inverse` | 反白文字 | `oklch(25% 0.04 240)` | 用于亮色卡片(如珊瑚色)上的深色字 |

### 品牌色与点缀色 (Brand & Accents)

为了在暗色背景上保持对比度和可读性，品牌暖珊瑚色需要稍微提高明度 (Lightness)。

| Token | 亮色模式 | 暗色模式 |
|-------|----------|----------|
| `--color-primary` | `oklch(76% 0.12 55)` 暖珊瑚 | `oklch(80% 0.14 55)` **提亮版珊瑚色** |
| `--color-primary-hover`| `oklch(70% 0.12 55)` | `oklch(85% 0.14 55)` |
| `--color-secondary` | `oklch(50% 0.04 240)` 蓝灰 | `oklch(75% 0.06 240)` 浅鸭蓝 |
| `--color-accent-sage` | `oklch(74% 0.10 130)` 鼠尾草 | `oklch(82% 0.12 130)` 提亮鼠尾草 |
| `--color-accent-lavender`| `oklch(74% 0.07 290)` 淡紫 | `oklch(84% 0.10 290)` 提亮淡紫 |
| `--color-accent-salmon`| `oklch(70% 0.14 25)` 鲑鱼粉 | `oklch(80% 0.16 25)` 提亮鲑鱼粉 |

---

## 暗色的阴影处理

在明亮的暖桃色下，我们的阴影是珊瑚色的。在深鸭蓝色的夜晚，阴影应当是**更深沉的鸭蓝色混合纯黑**。切忌使用白色发光替代阴影。

| Token | 暗色模式下的值 |
|-------|----------|
| `--shadow-sm` | `0 4px 12px oklch(15% 0.04 240 / 0.4)` |
| `--shadow-md` | `0 12px 32px oklch(15% 0.04 240 / 0.6)` |
| `--shadow-lg` | `0 24px 64px oklch(15% 0.04 240 / 0.8)` |
| `--shadow-glow` | `0 0 32px var(--color-primary)` (这里依然可以保留珊瑚色发光用于按钮) |

---

## 呼吸与切换 (Transitions)

由于 `07-motion-physics.md` 中定义的慢节奏，我们在亮暗模式切换时，也应该赋予它一种“日夜交替”的缓慢感。

```css
html {
  /* 使用极长的过度时间模拟日夜交替 */
  transition: background-color 2s ease-in-out, 
              color 2s ease-in-out;
}

@media (prefers-color-scheme: dark) {
  :root {
    color-scheme: dark;
    --color-background: oklch(25% 0.04 240);
    --color-surface: oklch(30% 0.05 240);
    /* ... 注入上方定义的其他暗色 Token ... */
    --shadow-md: 0 12px 32px oklch(15% 0.04 240 / 0.6);
  }
}
```

> **注意**：如果不希望用户在打开页面时看到闪烁，可以将 transition 应用在某个具体的容器类上，或者在 JavaScript 初始化完成后再添加 transition 类。
