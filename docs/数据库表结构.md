# 数据库表结构

## 技术说明

- 数据库：SQLite
- ORM：SQLModel（基于 SQLAlchemy）
- `Column(JSON)` 类型字段在 SQLite 底层存储为 TEXT，通过 SQLAlchemy 自动完成 JSON 序列化/反序列化

---

## 1. user（用户表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `uid` | str | PK | 用户唯一标识 |
| `username` | str | NOT NULL, 1-64 字符, 索引 | 用户名 |
| `identifier` | str | NOT NULL, 3-128 字符, 唯一索引 | 登录标识（手机号/邮箱/第三方 ID） |
| `provider` | str | 默认 "password", 索引 | 认证提供方（password / 第三方登录） |
| `password_hash` | str | 可空, 最大 128 | 密码哈希（provider=password 时使用） |
| `is_active` | bool | 默认 True | 账号是否激活 |
| `created_at` | datetime | UTC | 创建时间 |
| `updated_at` | datetime | UTC | 更新时间 |
| `last_login_at` | datetime | 可空 | 最后登录时间 |

---

## 2. userprofile（用户画像表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `user_uid` | str | PK, FK → user.uid | 关联用户，一对一 |
| `profile_data` | JSON | 默认 {} | 用户画像数据（完整 JSON 对象） |
| `profile_text` | str | 默认 "" | 用户画像的自然语言描述（从 profile_data 提取，用于检索） |
| `created_at` | datetime | UTC | 创建时间 |
| `updated_at` | datetime | UTC | 更新时间 |

### profile_data 结构

```json
{
  "type": "basic_profile",
  "stage": "generated",
  "question_mode": "none",
  "confirmed_info": {
    "current_grade": "字符串",
    "major": "字符串",
    "learning_stage": "字符串",
    "has_clear_goal": "字符串",
    "learning_method_preference": "字符串",
    "learning_pace_preference": "字符串",
    "content_preference": ["字符串数组"],
    "need_guidance": "字符串",
    "knowledge_foundation": "字符串",
    "strengths": "字符串",
    "weaknesses": "字符串",
    "experience": "字符串",
    "short_term_goal": "字符串",
    "long_term_goal": "字符串",
    "weekly_available_time": "字符串",
    "constraints": "字符串"
  },
  "defaulted_fields": ["字符串数组"],
  "question_md": "字符串",
  "question_box": {
    "question": "字符串",
    "options": [
      {
        "label": "用户可见选项文字",
        "value": "标准化选项值",
        "description": "选项说明",
        "target_fields": ["该选项影响的字段或动作"],
        "fills": {
          "字段或动作": "选择后对应的标准值"
        }
      }
    ]
  },
  "text": "自然语言描述全文"
}
```

| 嵌套字段 | 说明 |
|----------|------|
| `confirmed_info` | 用户确认的个人信息键值对 |
| `defaulted_fields` | 使用默认值的字段名列表 |
| `question_md` | 追问问题的 Markdown 文本 |
| `question_box` | 追问问题的交互式问答结构 |
| `question_box.question` | 问题文本 |
| `question_box.options` | 可选答案对象列表，包含 `label`、`value`、`description`、`target_fields`、`fills` |
| `text` | 用户画像的自然语言全文描述 |

## 3. userdifyconversation（用户 Dify 会话关联表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `user_uid` | str | PK, FK → user.uid | 关联用户，一对一 |
| `intent_conversation_id` | str | 默认 "" | 意图识别 Dify 会话 ID |
| `profile_conversation_id` | str | 默认 "" | 画像构建 Dify 会话 ID |
| `created_at` | datetime | UTC | 创建时间 |
| `updated_at` | datetime | UTC | 更新时间 |

---

## ER 关系

```
user (1) ──── (1) userprofile          (通过 user_uid 关联)
user (1) ──── (1) userdifyconversation  (通过 user_uid 关联)
```
